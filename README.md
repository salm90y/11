# نظام أرشفة الوثائق والكتب الرسمية مع محرك OCR العربي

تطبيق متكامل لإدارة وأرشفة الوثائق والكتب الرسمية الحكومية مع دعم استخراج النصوص العربية (OCR) أوفلاين وأونلاين.

## 📂 مجلدات ومكونات المشروع الرئيسية

1. **`python_backend/ocr_server.py`**:
   - سيرفر بايثون المحلي للاستخراج أوفلاين (Offline OCR Server).
   - يعتمد على معالجة الرؤية الحاسوبية بواسطة OpenCV (تكبير الدقة، تعديل ميلان الصفحة المسحوبة Deskewing، المعالجة الثنائية التكيفية).
   - يتضمن **المعجم العربي المحلي (Arabic Post-Processing Dictionary Engine)** ومحركات EasyOCR, Surya OCR, Tesseract.
   - يعالج ويصلح أخطاء مد الحروف ببرنامج Word (مثل اشتباه مد الحروف بحرف السين).

2. **`src/components/DocumentUpload.tsx`**:
   - واجهة أرشفة الوثائق واستخراج النصوص.
   - يدعم عرض النصوص بخط الوورد الافتراضي (Traditional Arabic) واتجاه الكتابة من اليمين لليسار.
   - يوفر خيار تحميل النص المفرغ كملف Word TXT جاهز للاستخدام.

3. **`server.ts`**:
   - سيرفر Node.js / Express الذي يدير التطبيق.

---

## 🚀 كيفية تشغيل سيرفر بايثون المحلي (Python Backend)

1. **تثبيت المتطلبات في Python**:
```bash
pip install flask flask-cors pillow numpy opencv-python easyocr torch
```

2. **تشغيل السيرفر المحلي**:
```bash
python python_backend/ocr_server.py
```
سيعمل السيرفر على المنفذ المحلي: `http://localhost:5000/api/ocr`

---

## 🛠️ المميزات الرئيسية
- **دعم خطوط Word المطبوعة**: التعرف على نصوص الكتب المطبوعة ببرنامج Word والمسحوبة بالسكانر.
- **معجم عربي محلي**: تصحيح الكلمات المصطلحات الرسمية والحكومية أوفلاين تلقائياً.
- **تصدير للوورد**: زر مباشر لتحميل النص المفرغ بصيغة ملف نصي متوافق مع MS Word.
